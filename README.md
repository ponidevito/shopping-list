# Покупки

Простий PWA-застосунок для списків покупок на чистому JavaScript.

## Що вже є

- офлайн-робота через Service Worker;
- локальне збереження в `localStorage`;
- кілька списків покупок з автоматичною назвою за датою і часом;
- перейменування списків;
- додавання, редагування і видалення покупок;
- корзина з автоочищенням через 24 години;
- пошук по назвах списків і покупках;
- шеринг через Web Share API або копіювання в буфер;
- налаштування фону, розміру шрифту і сортування;
- `manifest.webmanifest` для подальшої PWA/Google Play інтеграції.

## Google sync

Файл `google-sync.js` містить адаптер-заготовку. Для реальної синхронізації потрібно додати OAuth Client ID, увімкнути Google Drive API і реалізувати `pushSnapshot()` / `pullSnapshot()` через App Data Folder або окремий файл у Drive.

## Запуск

Для коректної роботи service worker відкрийте застосунок через локальний сервер, наприклад:

```bash
npx serve .
```
